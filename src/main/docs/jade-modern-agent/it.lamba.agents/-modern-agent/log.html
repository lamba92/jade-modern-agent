log

fun log(message: String, stream: Int = 1): Unit

Allows to log a message with a header that identifies the agent. By default log will print in System.out.

Parameters

message - The message to be printed.

stream - The output stream where the message will be printed. STD_OUT prints in System.out, STD_ERR prints in System.err. Default is STD_OUT.
